# Raffle Page

Aplicação de sorteio de números criada com HTML, CSS e JavaScript. O usuário informa um intervalo mínimo e máximo, e o sistema sorteia um número aleatório dentro desse intervalo.

## Visualização

Para visualizar o projeto localmente, basta abrir o arquivo `index.html` no navegador.

## Funcionalidades

- Entrada para número mínimo
- Entrada para número máximo
- Sorteio de número aleatório dentro do intervalo informado
- Validação para campos vazios
- Validação para garantir que o número máximo seja maior que o mínimo
- Exibição do resultado na tela
- Feedback sonoro para erro e resultado
- Botão atualizado para refazer o sorteio
- Layout responsivo para telas menores

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Como Executar

```bash
git clone https://github.com/joilsomichalski/raffle-page.git
cd raffle-page
```

Depois, abra o arquivo `index.html` no navegador.
